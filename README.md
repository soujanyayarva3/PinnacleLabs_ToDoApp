# 📋 Task Manager Pro

A modern and responsive task management application built using HTML, CSS, and JavaScript. The application helps users organize daily activities, track task progress, and improve productivity through a clean and intuitive interface.

## 🚀 Features

* Add new tasks instantly
* Mark tasks as completed
* Delete tasks
* Search tasks in real time
* Task statistics dashboard

  * Total Tasks
  * Completed Tasks
  * Pending Tasks
* Local Storage support (tasks persist after page refresh)
* Responsive and modern user interface
* Glassmorphism-inspired design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage API

## 💡 How It Works

1. Enter a task in the input field.
2. Click **Add Task** to save it.
3. Click **✔** to mark a task as completed.
4. Click **✕** to remove a task.
5. Use the search bar to quickly find tasks.
6. Task data is automatically saved in the browser using Local Storage.
