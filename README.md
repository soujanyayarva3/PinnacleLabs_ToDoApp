# PinnacleLabs_ToDoApp
To-Do List App
Simple To-Do List Application using HTML, CSS, and JavaScript

Overview
This project is a simple To-Do List web application that helps users add, manage, and delete daily tasks.

Objective
To build a task management web app using frontend development skills.

Features
Add new tasks
Delete tasks
Mark tasks as completed
Simple and clean UI
Responsive design

Technologies Used
HTML
CSS
JavaScript

Project Workflow
Create UI layout using HTML
Style the interface using CSS
Add functionality using JavaScript
Handle add, delete, and update tasks
Test functionality in browser

Result
The application allows users to manage their daily tasks efficiently in the browser.
